# PlayStation 5 x Cleverbridge

This website is an informative and accessible web project about the PlayStation 5 and its multimedia technologies, created for Cleverbridge. The platform showcases the PS5’s technological highlights and hardware specifications, offers an interactive multimedia demo to illustrate its audio and video capabilities, and explains practical applications and technological innovations that shape the modern gaming experience.

Demo: https://imanhri.github.io/PS5xCleverBridge/
